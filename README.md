# POYECTO_FINAL_PDS2
Proyecto final del curso de Procesamiento Digital de Señales II

Descripción General

Este proyecto implementa un sistema de visión por computador que digitaliza automáticamente la indicación de presión de manómetros industriales analógicos a partir de una imagen fotográfica ordinaria.

El sistema combina:
-Transformada de Hough Circular + ajuste por Mínimos Cuadrados → detección precisa del dial
-Transformada de Hough Lineal Probabilística + filtro geométrico → localización de la aguja
-Modelo de geometría espacial → conversión del ángulo en valor numérico de presión
